A modern, professional redesign of the Tax Home Kenya Limited website with enhanced features, improved design, and new sections.

## 🎯 Project Overview

This is a complete redesign of the Tax Home Kenya website featuring:

- **Modern Design**: Premium, elegant look with professional color scheme and smooth animations

## 🚀 Getting Started

### Installation

1. Clone the repository or download all files
2. Ensure the folder structure matches the layout above
3. Replace `assets/hero.jpg` with the actual hero image from the original site
4. Upload all files to your GitHub Pages repository

### Local Testing

1. Open `index.html` in a web browser
2. Navigate through all pages to verify functionality
3. Test responsive design by resizing the browser window
4. Test on mobile devices

### Deployment to GitHub Pages

1. Push all files to your `taxhomekenya.github.io` repository
2. Ensure the repository is set to public
3. GitHub Pages will automatically deploy the site
4. Visit `https://taxhomekenya.github.io` to view the live site

## 🔧 Customization

### Updating Content

- **Homepage**: Edit `index.html` directly
- **Blog Posts**: Edit files in `pages/` directory
- **Styles**: Modify `css/styles.css` for design changes
- **Functionality**: Edit `js/main.js` for interactive features

### Adding New Blog Posts

1. Create a new HTML file in the `pages/` directory
2. Use the same template structure as existing blog posts
3. Add a link to the new post in `pages/blog.html`
4. Update the homepage blog section if featuring the new post

### Adding New Resources

1. Create or prepare your PDF file
2. Place it in the `resources/` directory
3. Add a resource card in `index.html` with a download link
4. Update the resources section as needed
